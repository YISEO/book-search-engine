# Book Search Engine

## Description
This is a book search engine application built with a GraphQL API using Apollo Server. The app allows users to search for books, save them to their account, and manage their saved book list.

## Installation
If you'd like to run this application locally or make changes, follow these steps:

1. Clone this repository.
2. Navigate to the project directory.
3. Install the required dependencies using `npm install`.
4. Start the development server with `npm start`.

## Features
- Search for Books: Users can search for books using the Google Books API.
- User Authentication: Users can sign up or log in to the application.
- Save and Remove Books: Authenticated users can save books to their account and remove them.
- View Saved Books: Users can view a list of books they have saved.

## Usage
- Open your web browser and go to http://localhost:3000.
- The application presents a menu with options to search for books and login/signup.
- When you click on "Search for Books," you can enter a search term and view search results.
- If you are not logged in, you can still search for books and view details.
- To login or sign up, click on "Login/Signup" in the menu.

## Deployment
The application is deployed on Heroku with a MongoDB database using MongoDB Atlas. 
You can view [a live demo](https://peaceful-caverns-23899-938a562d6b18.herokuapp.com/)
